# Remixes

## How to create a Remix

Create a fork fo the original repository, click the fork button

Clone your forked repo on your Github profile
```git clone https://github.com/YOUR_USERNAME/Fail-safe```

Make a copy of the Fail-safe.js to remixes/ folder with your new name!
Example-Remix-Fail-safe.js

Make your changes

Now stage your changes
```git add Example-Remix-Fail-safe.js```

Create a commit message
```git commit -m "My super cool remix"```

Push your change
```git push```

Create a PR (Pull Request)
Find your fork on github, and Contribute by Creating a Pull Request
